Pixel icon from the HackerNoon Pixel Icon Library (line + `solid` styles). Inherits `currentColor`, sized by `size`.

```jsx
<Icon name="search" />
<Icon name="robot" solid size={24} color="var(--violet-500)" />
<Icon name="chart-line" title="Analítica" />        // accessible
<IconButton label="Buscar"><Icon name="search" /></IconButton>
```

Names are the `hn-<name>` glyphs (e.g. `home`, `cog`, `bell`, `chart-line`, `sparkles`, `lock`, `user`). Full set in the Icons card / `tokens/icons.css`.
