Square icon-only control for toolbars and dense actions. Always pass `label`.

```jsx
<IconButton label="Buscar"><SearchIcon/></IconButton>
<IconButton label="Más" variant="outline"><MoreIcon/></IconButton>
<IconButton label="Enviar" variant="solid"><ArrowIcon/></IconButton>
```

Variants: `ghost`, `outline`, `solid`. Sizes `sm`/`md`/`lg`. Pass a Lucide SVG as children.
