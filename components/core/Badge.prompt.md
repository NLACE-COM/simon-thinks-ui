Compact status / metadata label. Use `accent` for module/feature tags, semantic variants for state, `dot` for live indicators.

```jsx
<Badge variant="accent">CORE</Badge>
<Badge variant="success" dot>Activo</Badge>
<Badge variant="neutral">v2.4.0</Badge>
```

Variants: `neutral`, `accent` (violet soft), `solid` (violet), `success`, `warning`, `danger`. Prop: `dot`.
