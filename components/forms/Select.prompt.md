Native select styled to match Input. Pass `options` (strings or `{value,label}`).

```jsx
<Select label="Módulo" options={['THINK','CORE','CUSTOM','LIGHT']} />
<Select label="Plan" placeholder="Elegir…" options={[{value:'pro',label:'Pro'}]} />
```

Props: `label`, `hint`, `error`, `options`, `placeholder`, plus native select props.
